# performance-difficulty-estimation

Набор инструментов для оценки успеваемости студентов и сложности учебных заданий с использованием различных статистических подходов.

## Описание

Этот проект предоставляет три различных метода оценки параметров:
- **MAP (Maximum a Posteriori)** - байесовский подход с поиском максимума апостериорного распределения
- **MCMC (Markov Chain Monte Carlo)** - байесовский подход с семплированием из апостериорного распределения
- **MLE (Maximum Likelihood Estimation)** - частотный подход с максимизацией функции правдоподобия

## Методы

### MAP Estimator
- Maximum a Posteriori
- Априорное распределение: Beta(3, 3)

### MCMC Estimator
- Markov Chain Monte Carlo
- Априорное распределение: Beta(3, 3)
- 20 семплов (можно увеличить для реальных задач)

### MLE Estimator
- Maximum Likelihood Estimation
- Метод оптимизации: Powell

## Формат данных

Данные должны быть представлены в виде матрицы размера (students_count × units_count), где:

- **Строки**: студенты
- **Столбцы**: юниты
- **Значения**:
  - `1` - юнит сдан
  - `0` - юнит не сдан
  - `NaN` - студент не начинал сдавать юнит
